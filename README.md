# Algorithm_Study
SSAFY 알고리즘 스터디


⭐진행 방식

[월요일~금요일 (온라인)]
SWEA, 백준 알고리즘 5문제 풀이 -> 핵심 로직, 어려웠던 점 공유 -> 코드 리뷰 시행

[토요일/일요일 택1 (오프라인)]
제한 시간(1시간) 내 SWEA, 백준 알고리즘 2문제 풀이 -> 핵심 로직, 어려웠던 점 공유 -> 코드 리뷰 시행


⭐규칙
1. 핵심 로직, 어려웠던 점을 중심으로 자유롭게 주석을 작성한다.
2. 2월은 백준 그룹에 있는 입문, IM, A형 문제를 위주로 풀이한다.
3. 매주 정해준 문제를 제한 일자까지 반드시 풀이한다.
